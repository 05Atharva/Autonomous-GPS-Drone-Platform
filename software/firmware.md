# Firmware Overview

The drone operates on a flight controller firmware compatible with Mission Planner (Radiolink).

## Firmware Responsibilities

- Sensor fusion (IMU, GPS, Barometer)
- Flight stabilization
- Autonomous flight mode execution
- Waypoint navigation
- Failsafe handling (RTL)

## Configuration Approach

- Firmware configured through Mission Planner
- No source-level firmware modifications
- Emphasis on stability, tuning, and safety
