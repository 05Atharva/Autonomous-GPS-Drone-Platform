# Hardware Overview

This directory documents the **physical hardware components and subsystems**
used in the Autonomous GPS-Based Drone Platform.

The focus here is on **system-level understanding** rather than detailed
assembly instructions.

## Covered Topics

- Major airframe and propulsion components
- Flight controller and navigation sensors
- Custom telemetry system design
- Role of each hardware block in autonomous flight

⚠️ This repository does not include step-by-step build or wiring instructions.
