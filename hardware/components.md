# Hardware Components

## Airframe & Propulsion
- Frame: S500 quadcopter frame
- Motors: 850KV Avionic BLDC motors
- ESCs: 30A ESCs

## Flight Control
- Flight Controller: Crossflight FC

## Navigation & Sensors
- GPS Module: u-blox NEO-M8
- Barometer: GPS module integrated barometer
- IMU: Flight controller onboard sensors

## Communication
- Telemetry: Custom HC-12 based telemetry kit
- Configuration MCU: Arduino Uno (for HC-12 AT commands)

## Radio Control
- Receiver: FlySky FS-i6XB
- Signal Modes: PPM / PWM
