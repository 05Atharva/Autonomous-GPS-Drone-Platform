# Autonomous GPS-Based Drone Platform

An autonomous quadcopter platform developed as a club project, capable of
**GPS-based navigation, waypoint missions, and Return-to-Launch (RTL) failsafe**.

This project focuses on **hardware integration, flight-controller configuration,
telemetry design, and autonomous flight logic** rather than custom flight firmware.
